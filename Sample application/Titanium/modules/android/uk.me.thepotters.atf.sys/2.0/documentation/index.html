atfsys Module

Description

Part of my in-house framework this very simple module contains a few Android system utilities (Kill My Process & Force Garbage Collection). I find these essential when producing apps for Android.

Accessing the atfsys Module

To access this module from JavaScript, you would do the following:

var atfsys = require("uk.me.thepotters.atf.sys");

The atfsys variable is a reference to the Module object.

Reference

Only two functions supported at the moment:

OptimiseMemory()

Triggers the Force Garbage collection thread within the module. There is a 25ms debounce on this function to prevent multiple calls from triggering too many garbage collects.

KillMyProcess()

Calls into Android kernel to force termination of the process hosting your activity. It is strongly recommended that you call the 'finish' method of the current activity before calling this function.

Usage

// Include module var atfsys = require('uk.me.thepotters.atf.sys');

// Optimise memory i.e. Force Garbage collection atfsys.OptimiseMemory();

// Kill our process right now atfsys.KillMyProcess();
